What is the book value?
The book value is the difference between the company's total assets and total liabilities. The book value present's the total value of the company's assets that the shareholders of this company would receive if the company liquidaged. How to calculate book value?
To calculate book value you should use the following formula:
book value = total assets - total liabilities
example: company x has $6,700,000 in total assets and $5,100,000 worth of total libilities, calculate the book value?
book value = total assets - total liabilities
book value = $6,700,000 - $5,100,000
book value = $1,600,000
Why use book value?
The book value is used to calculate the value of a firm's total assets that the shareholders would receive if the firm was liquidated.
